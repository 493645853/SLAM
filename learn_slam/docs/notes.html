Coordinates & Basis

A point in the real Cartesian space R3 can be described by the basis (e1,e2,e3), as

(1)a=[e1,e2,e3][a1a2a3]=a1e1+a2e2+a3e3.

Inner/Outer Product

For two vectors a,b∈R3, their inner product is defined as

(2)a⋅b=|a||b|cos⁡⟨a,b⟩

and their outer product is

(3)a×b=[ijka1a2a3b1b2b3]=[a2b3−a3b2a3b1−a1b3a1b2−a2b1]=[0−a3a2a30−a1−a2a10]b≜a∧b

which is orthogonal to the vectors a and b. Here, a∧ is a "Skew-symmetric" (or anti-symmetric) matrix.

Euclidean Transformation

Suppose the world coordinates (xw,yw,zw) are stationary while the robot can be indicated by a moving coordinates (xc,yc,zc). Consider a vector p∈R3 in the figure below: We can represent the vector using those two different coordinates. Assume the world coordinates are described by the basis (e1,e2,e3) and the robot coordinates are described by the basis (e1′,e2′,e3′), the vector p will not change using the representations from those two bases, i.e.,

(4)[e1,e2,e3][a1a2a3]=[e1′,e2′,e3′][a1′a2′a3′].

Then, multiply the two sides with [e1,e2,e3]T,

(5)[a1a2a3]=[e1,e2,e3]T[e1′,e2′,e3′][a1′a2′a3′]≜Ra′.

where R denotes the rotation matrix, which contains certain special properties. We can define the set of the rotation matrix as

(6)SO(n)={R∈Rn×n|RRT=I,det⁡(R)=1}

where SO(n) stands for "Special Orthogonal Group". Basically, the rotation matrix can describe the rotation of certain object. We can also perform the inverse operation (the reversed rotation) by R−1a (equivalent to RTa since R is symmetric) to obtain the vector a′.

Moreover, the Euclidean transformation also includes the translation of the robot's location, and thus we also need the translation vector t∈R3 to the original expression, i.e.,

(7)a′=Ra+t.

Overall Transform Matrix

We can represent above Euclidean transformation into a homogeneous coordinates form, as

(8)[a′1]=[Rt0T1][a1]≜T[a1]

which allows us to represent the overall transform in a linear form, and in the rest of the notes, I will implicitly represent the homogeneous coordinates [a,1]T by a for simplicity. In addition, the set of the transform matrix T can be defined as

(9)SE(3)={T=[Rt0T1]∈R4×4|R∈R3×3,t∈R3}

where denotes the special Euclidean group.

Rotation Vector & Euler Angles

In , we employ totally 16 entries to describe the Euclidean transform of the object, and in SO(3) we employ 9 entries to describe the rotation with 3 DoFs, which can be redundant. A more compact representation of the rotation transform can be

  • Rotation axis + rotation angle
  • Euler angles

which will be introduced as follows.

Rotation Vector

Suppose the rotation axis is given by the vector n, and the rotation angle is θ, the rotation vector can be represented by θn, which relates the rotation matrix using the Rodrigues’s Formula1, as

(10)R=cos⁡θI+(1−cos⁡θ)nnT+sin⁡θn∧

where ∧ is the operator that transform the vector to the anti-symmetric matrix. From this relation, we can also retrieve the rotation angle by

tr⁡(R)=cos⁡θtr⁡(I)+(1−cos⁡θ)tr⁡(nnT)+sin⁡θtr⁡(n∧)=3cos⁡θ+(1−cos⁡θ)(11)=1+2cos⁡θ

and therefore we can obtain the rotation angle by

(12)θ=cos−1⁡{tr⁡(R)−12}

In addition, if we rotate the rotation axis n by the matrix R, the result are still Rn=n. Therefore, n is the eigenvector of the matrix R which corresponds to the eigenvalue 1. We can use eigen decomposition to find the rotation axis.
